You can mix filetype if you have some images that need to be .png due to transparancy, for example if you want rounded corners.

3 modes are possible and will be activated automatically:
"No backs" is active if 0 backs are selected.
"Same backs" is active if 1 back is selected.
"Unique backs" is active if number of backs is same as fronts. They will be paired in the order you've sorted them, for example alphabetically.

Tip : If you want same back for all fronts be sure to only select 1 back image. This will reduce the filesize because only 1 image is "loaded" into the pdf even if its displayed on all pages. If you have big groups of same backed cards its better to run them each seperatly and later combine the pdf's.

Pick the page size and orientation you want to use. Note that some presets may require certain orientation. If your grid layout and card size dont fit on the page you will get a warning.

The preset loadouts are common setups, you still need to pick your page size but all other settings are adjusted. You can ofcourse "fine tune" all the presets to your requirements.

Basic settings

Specify how you want the grid layout to look, how many rows/columns. Grid is filled top to bottom and left to right (right to left on even pages), flip horizontaly to print double sided.

Grid layout

All measurements are in millimeters(mm). Card width/height refers to the final card size, bleed is added to that on all sides. Example, you want a card that is 63x88mm with 3mm bleed. Your image files should be 69x94mm (or that ratio)

Measurements

Advanced settings

Cut lines are in the style of crosshairs (small cross on each card corner). If you print doublesided i recommend only having crosshairs on one side. Here you can pick if you want them on front/back and you can also customize the line width, total size and the color.

Crosshair settings

Border is a great option to add to battle any alignment issue. If border is activated it will shrink the card image for that side and fill the space with a color of your choice. It will also add a few extra mm around the outside of the grid (for alignment issues).

Border settings

Tip : If your card design already have a solid color(or close to) border you can still add border and set the [width: 0]. This will just add an extra border to the edges of the grid and help with alignment.

Tip 2 : Personaly i prefer to have the card image with rounded corners if im using a border. You can achive this by rounding the corners of your image and saving it as a .png with transparent background. It will take a bit longer to generate but its worth it. Remember to only select 1 back if you want to use the same for all fronts.

Click Generate PDF to start the process. As long as you see the green loading circle it should be processing, except if you've gotten an error, for example the grid not fitting on the page.
Remember 1000+ .jpg only takes seconds, 100 .png takes about 30sec, so if nothing happens in in a resonable time and you dont see any warnings, just refresh the page, an unknown error might have happened.

Tip : Images stay selected after you've generated a pdf, you can go back and change things and generate again to quickly try different settings.
